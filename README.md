# Slimer - slime's war has begun
"Slimer" est un jeu par navigateur gratuit et sans attentes intempestives : https://les-jeux-de-bebel.itch.io/slimer
Qui a ete devellope a  l'issus de la SlimeJam2000 de itch.io : https://itch.io/jam/slimejam2000

Il s'agit d'un clicker minimalist dans lequel le joueur doit exploser le plus de slime possible.

Bon test, et n'™hesitez pas a revenir vers nous pour plus d'informations.

Je vous invites egalement a aimer notre page Les Jeux de Bebel sur Facebook : https://www.facebook.com/lesjeuxdebebel/
